MCDU App Fenix A320 — Remote MCDU for MSFS 2020 & 2024
A simple desktop app (Windows & macOS) to use the Fenix A320 MCDU on a second screen, tablet, or any device on your network.
What it does
This app connects to the Fenix A320's built-in web server (port 8083) and shows the MCDU in its own clean window. No extra setup, no plugins — it uses Fenix's native system.
Features

Waits for Microsoft Flight Simulator to be ready, then connects automatically
Works on the same PC (localhost) or on a different PC over your home network — just enter the IP address
Choose Captain (left) MCDU, First Officer (right) MCDU, or show both side by side
Both MCDUs displayed together with a clean center divider
Quick dropdown menu to switch between CPT, FO, and Both views at any time
Live connection status so you always know when the sim is ready

How to use

Start MSFS and load into the Fenix A320
Pop out the MCDU display in the cockpit (so Fenix can serve it)
Open the app and enter your sim PC's IP address (or use localhost if it's the same PC)
Pick the MCDU you want — and you're flying

Requirements

Fenix A320 (Block 2 or later, with the EFB web server enabled)
Microsoft Flight Simulator 2020 or 2024
Windows or macOS
